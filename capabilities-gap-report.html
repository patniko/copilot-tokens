Custom Tool Definitions High Impact
The SDK's defineTool() lets you register custom tools with typed Zod schemas that the AI can invoke. This is the single most powerful showcase feature â€” you could add Electron-native tools (notifications, clipboard, screenshot, system info, open apps, file dialogs) that demonstrate the SDK's extensibility far beyond the built-in CLI tools.

Session Lifecycle Hooks High Impact
Six powerful hooks let you intercept and modify the agent's behavior at every stage: modify tool args before execution, inject context after tool results, rewrite user prompts, handle errors with retry logic, and run cleanup on session end. None are wired up.
onPreToolUse onPostToolUse onUserPromptSubmitted onSessionStart onSessionEnd onErrorOccurred

When the agent uses the ask_user tool, the SDK fires onUserInputRequest. Currently this goes unhandled â€” the agent can't ask clarifying questions. Wiring this up would show interactive multi-turn Q&A with choice selection in the UI.
